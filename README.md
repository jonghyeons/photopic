## Photopic

JPG, RAW file sorter program in terminal written golang



### How to install

```bash
$ git clone https://github.com/jonghyeons/photopic.git
$ cd photopic
$ go install
```



### Usage

```bash
# Run in current filepath
$ photopic

# Run another filepath
$ photopic YOURFILEPATH
```



### Directory after runing

```bash
+-- /FILEPATH
    +-- /jpg
    +-- /raw
```
